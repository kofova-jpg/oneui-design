Preference based SeekBarPreference but with more options and flexibility

If isAdjustable is set to true, shows + and - buttons at both ends of the seekbar

var centerBasedSeekBar: Boolean

If true, this sets SeslSeekBar to only put tick marks at the start, middle and end regardless of the min and max value. This will ignore SeslSeekBar.setMode value and will use MODE_LEVEL_BAR mode instead.
