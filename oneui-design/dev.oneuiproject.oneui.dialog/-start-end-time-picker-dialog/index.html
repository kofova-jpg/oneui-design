StartEndTimePickerDialog

class StartEndTimePickerDialog(context: Context, startTime: Int, endTime: Int, is24HourFormat: Boolean = DateFormat.is24HourFormat(context), timePickerChangeListener: StartEndTimePickerDialog.TimePickerChangeListener?) : AlertDialog, SeslTimePicker.OnTimeChangedListener, SeslTimePicker.OnEditTextModeChangedListener

A dialog that allows users to pick a start and end time using a tabbed interface.

This dialog presents two tabs (Start and End), each with a time picker. The user can switch between the tabs to set the start and end times independently. The dialog supports both 24-hour and 12-hour formats.

Parameters

context

The context to use for building the dialog.

startTime

The initial start time in minutes from midnight (0-1439).

endTime

The initial end time in minutes from midnight (0-1439).

is24HourFormat

Whether to use 24-hour time format. Defaults to the system setting.

timePickerChangeListener

Listener to receive the selected start and end times when the user confirms.
