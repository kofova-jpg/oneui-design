AppInfoLayout

class AppInfoLayout(context: Context, attrs: AttributeSet?) : ToolbarLayout(source)

Custom App Info Layout like in any App from Samsung showing the App name, app version, and other related app info.

Important:

  • This view must be hosted within an AppCompatActivity as it relies on AppCompat-specific features and theming. Otherwise, it will result to runtime exceptions.
