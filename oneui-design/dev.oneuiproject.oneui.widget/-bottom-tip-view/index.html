BottomTipView

class BottomTipView @JvmOverloads constructor(context: Context, attrs: AttributeSet? = null, defStyleAttr: Int = 0, defStyleRes: Int = 0) : RoundedLinearLayout(source)

A custom view that displays a tip at the bottom of the screen.

This view consists of a title, a summary, and an optional link. The title, summary, and link text can be set programmatically or through XML attributes.

Example usage:

<dev.oneuiproject.oneui.widget.BottomTipView
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    app:title="Tip Title"
    app:summary="This is a helpful tip."
    app:linkText="Learn More" />
