RadioItemViewGroup

class RadioItemViewGroup @JvmOverloads constructor(context: Context, attrs: AttributeSet? = null) : LinearLayout(source)

A ViewGroup that displays a set of RadioItemViews vertically.

This layout arranges its children in a single vertical column. The RadioItemViews within this group are mutually exclusive; selecting one RadioItemView will automatically deselect any previously selected item.

This behavior is similar to a standard RadioGroup but is designed specifically for custom RadioItemView instances.
