MarginsTabLayout

open class MarginsTabLayout @JvmOverloads constructor(context: Context, attrs: AttributeSet? = null, defStyleAttr: Int = com.google.android.material.R.attr.tabStyle) : TabLayout(source)

An extension of TabLayout that dynamically adjusts its width to fit the parent by setting appropriate side margins. It ensures that the selected tab is always visible by automatically scrolling to it.

This class provides a mechanism to customize the tab dimensions through the setCustomTabDimen method, allowing for flexible layout adjustments.
