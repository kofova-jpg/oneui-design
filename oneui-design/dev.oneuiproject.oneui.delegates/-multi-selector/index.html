MultiSelector

interface MultiSelector<T>(source)

Interface for implementing multi-selection in a RecyclerView.Adapter.

This interface provides methods for configuring multi-selection, managing the selection state, and interacting with the selection mode.

Parameters

T

The type of the selection ID. This should typically be a stable identifier for your items, such as Long when using RecyclerView.Adapter.getItemId.
